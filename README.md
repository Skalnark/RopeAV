# Rope Algorithm Visualizer

This project is a visualizer for the Rope data structure algorithm. It provides an interactive way to understand how ropes work, including operations like concatenation, splitting, and substring extraction.

We want to met the same features the [Bloom Filter Visualizer](https://skalnark.github.io/bloom_filter_algorithm_visualizer/index.html) has, so we have the following goals:

- [ ] Step by step execution of each operation with explanations.
  - [ ] Building the Rope from a string.
  - [ ] Inserting a substring (concatenation).
  - [ ] Deleting a substring (splitting).
  - [ ] Extracting the string (toString operation).
- [ ] Interactive visualization of the Rope data structure.
- [ ] Step-by-step demonstration of Rope operations (concatenation, splitting, substring extraction).
- [ ] User interface to input strings and see how they are represented as ropes.
- [ ] Educational content explaining the Rope data structure and its advantages.
- [ ] Responsive design for accessibility on various devices.

The main problem here is adding the step-by-step execution of each operation with explanations, as it requires breaking down each operation into smaller steps and keeping track of the state of the Rope at each step.
